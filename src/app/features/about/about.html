Info

Gotoshi is the first ever existing Blockchain game that runs only in a browser with no centralized backend server.
It connects directly to Bitcoin full nodes for sending data other players.

You can read the thesis here: Peer-to-peer gaming on the Bitcoin Blockchain.

You can also hire me!

In the navbar you see:

Blockchain connection state

This is the current state of your connection to the Blockchain network. Left is the current block heigth and right the time that this block is behind. The connection is established with a websocket and the help of Webcoin.

Latest transaction ID

Here you find the latest transaction ID and on click a overview of the wallets unspend transaction outputs (UTXOs).

Wallet address & balance

This is the wallet address that is used to spend funds to the game wallet and represents the player in the games. Click for full address.

Instructions

To connect to a local socket instead of mine install the websocket-to-tcp-bridge:
npm install -g webcoin-bridge
npm install -g webcoin-bitcoin-testnet
webcoin-bridge --network webcoin-bitcoin-testnet --port 8193
When you want to debug you can setup debug in the console:
localStorage.debug = 'gotoshi:*'
or for the full log
localStorage.debug = '*'
