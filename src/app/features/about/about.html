Instructions

Gotoshi is the first ever existing Blockchain game that runs only in a browser with no backend server.
It connects directly to Bitcoin full nodes for sending data other players.

To play please install a local bridge:
npm install -g webcoin-bridge
npm install -g webcoin-bitcoin-testnet
webcoin-bridge --network webcoin-bitcoin-testnet --port 8193

Debug with console:
localStorage.debug = 'gotoshi:*'
or for the full log
localStorage.debug = '*'
